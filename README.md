# 📊 Central Tendency Insights

## 🚀 Overview
This project focuses on analyzing a dataset using fundamental statistical measures — **Mean, Median, and Mode**.

The goal is simple:
Not just calculate numbers, but understand what they actually say about the data.

---

## 🎯 Objectives
- Apply central tendency concepts on real data  
- Compare Mean, Median, and Mode  
- Understand the impact of outliers  
- Interpret data distribution using statistics  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---



## 📊 Analysis Performed
- Data loading and cleaning  
- Calculation of:
  - Mean  
  - Median  
  - Mode  
- Visualization using plots  
- Comparison of results  
- Interpretation of distribution  

---

## 📈 Key Insights
- Mean is heavily affected by extreme values (outliers)  
- Median gives a more stable central value in skewed data  
- Mode highlights the most frequent occurrence  
- Real-world data is messy — choosing the right measure matters
